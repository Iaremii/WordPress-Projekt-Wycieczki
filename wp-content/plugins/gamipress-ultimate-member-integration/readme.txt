Connect GamiPress with Ultimate Member

== Description ==

Gamify your [Ultimate Member](http://wordpress.org/plugins/ultimate-member/ "Ultimate Member") community thanks to the powerful gamification plugin, [GamiPress](https://wordpress.org/plugins/gamipress/ "GamiPress")!

This plugin automatically connects GamiPress with Ultimate Member adding new activity events and features.

= New Events =

* Account approved: When an user account gets approved.
* Account marked as inactive: When an user account gets marked as inactive.
* Change profile photo: When an user changes the profile photo.
* Change cover photo: When an user changes the cover photo.
* Remove profile photo: When an user removes the profile photo.
* Remove cover photo: When an user removes the cover photo.
* Update profile description: When an user updates the profile description.
* Update account information: When an user updates the account information.

= New Features =

* Drag and drop settings to select which points types, achievement types and/or rank types should be displayed on user frontend profiles and in what order.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
